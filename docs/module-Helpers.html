(static) em(pxval, baseopt) → {String}

Source:
  • helpers/em.js, line 31

Convert pixel value to ems. The default base value is 16px, but can be changed with the second argument

Example
// Styles as object usage
const styles = {
  'height': em('16px')
}

// styled-components usage
const div = styled.div`
  height: ${em('16px')}
`

// CSS in JS Output

element {
  'height': '1em'
}
Parameters:
Name Type Attributes Default Description
pxval string | number

The pixel value you want to convert
base string | number <optional>
 "16px"

The base size to convert from
Returns:

The converted value

(static) stripUnit(value) → {Number}

Source:
  • helpers/stripUnit.js, line 27

Strip the unit from a given CSS value

Example
// Styles as object usage
const styles = {
  '--dimension': stripUnit(100px)
}

// styled-components usage
const div = styled.div`
  --dimension: ${stripUnit(100px)}
`

// CSS in JS Output

element {
  '--dimension': 100
}
Parameters:
Name Type Description

A string containing a value with its unit.
Returns:
